Pillow¶

Pillow 是 亚历克斯·克拉克和贡献者 对PIL友好分叉. PIL是由 Fredrik Lundh 和贡献者们开发的Python图像库.

通过 Tidelift Subscription 对 企业 Pillow 支持. 了解更多.
